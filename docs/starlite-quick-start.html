Starlite
Quick Start Guide
1 Basic Setup
1
Connect the USB-C Cable
Plug the USB-C cable into your Starlite device. Loop the cable around the nub for cable support to keep everything neat and secure.
2
Connect Power Adapter
Connect the other end of the USB cable to your power adapter and plug it into a power outlet.
3
Power On
Starlite should start right away with the default lights setup. Your LED strip will begin displaying the default effect immediately!
✨
Success!

Your Starlite is now powered on and running. You can use it with the default settings or proceed to Advanced Setup for full control.

2 Advanced Setup
📱 Mobile App Required

For advanced control and customization, you'll need the Kolori mobile app. Follow the steps below to get started.

1
Download Kolori App
Download the Kolori app from the official website:
Download Kolori App
2
Connect to Starlite Wi-Fi
Open your phone's Wi-Fi settings and look for the following access point:
Network Name: starlite
3
Enter Wi-Fi Password
Connect to the "starlite" network using one of the following passwords:
  • starlite4321 (Primary)
  • wled1234 (Fallback)
💡 Tip

Try the passwords in order. The first one (starlite4321) is most commonly used.

4
Add Device in Kolori App
Open the Kolori app and add your Starlite device manually using the following IP address:
IP Address: 4.3.2.1
How to Add Device

When you open the Kolori app for the first time, you'll see the onboarding page.
1. Tap "Add Device Manually"
2. Enter device name (e.g., "Starlite")
3. Enter IP address: 4.3.2.1
4. Tap "Save"

5
Configure Presets
Now you're ready to customize! Use the Kolori app to:
  • Create custom lighting effects
  • Set your favorite preset as boot preset
  • Create playlists that cycle through multiple effects
  • Adjust brightness and colors
  • Control your lights from anywhere on the network
🎉
All Set!

Your Starlite is now fully configured. Enjoy creating amazing lighting displays!

🔧 Troubleshooting
Can't find "starlite" Wi-Fi?
• Make sure Starlite is powered on
• Move closer to the device
• Restart Starlite by unplugging and replugging power

Can't connect to the Wi-Fi?
• Double-check the password (case-sensitive)
• Try all three password options
• Forget the network and reconnect

App can't find device at 4.3.2.1?
• Ensure you're connected to "starlite" Wi-Fi network
• Try the IP address in a web browser first
• Check your phone's Wi-Fi connection status
