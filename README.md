## Overview

A mobile app for controlling [WLED](https://kno.wled.ge/) LED devices. Real-time control, audio-reactive effects, and preset management for iOS and Android.

**Features:**
- ⚡ Real-time WebSocket control
- 🎵 Audio-reactive effects with FFT
- 📡 Live LED visualization
- 🎨 Presets and playlists
- 🌓 Dark/light themes

## Quick Start

```bash
git clone https://github.com/mrkprdo/kolori.git
cd kolori
npm install

# Run on device
npx expo run:android  # Android
npx expo run:ios      # iOS (macOS only)
```
